Card Matching Game

A simple and interactive Card Matching (Memory) Game built using HTML, CSS, and JavaScript as part of a vibe coding assignment.

Features
- 16 cards (8 matching pairs)
- Click to flip cards
- Match checking logic
- Move counter
- Win message when all pairs are matched
- Restart game button

Technologies Used
- HTML
- CSS
- JavaScript
- Replit Agent (AI-powered vibe coding tool)
